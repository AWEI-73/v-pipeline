# Local SFX Library

These six WAV files are deterministic synthetic assets generated with ffmpeg
for Hermes Video Pipeline. They contain no third-party recordings.

License: CC0 1.0 / public domain dedication.

Roles:

- `whoosh_1.wav`, `whoosh_2.wav`: chapter transitions
- `hit_1.wav`, `hit_2.wav`: explicit title-card entrances
- `riser_1.wav`, `riser_2.wav`: reserved for later structural cues

Keep mix volume restrained; S3a defaults to `0.15`.
