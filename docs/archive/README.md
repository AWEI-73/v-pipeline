# Docs Archive

This folder keeps historical records that should not be treated as the current
operating path unless an active document links to them for evidence.

Contents:

- `decisions/` - dated decision logs and acceptance reports.
- `reviews/` - historical review notes and evidence.
- `roadmap-history/` - superseded long-form roadmap snapshots.

Use active docs in `docs/` first, especially `START_HERE_VIDEO_PIPELINE.md`,
`video-pipeline-end-to-end-line.md`, and `video-pipeline-operating-map.md`.
