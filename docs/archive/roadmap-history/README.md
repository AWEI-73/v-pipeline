# Roadmap History Archive

This folder holds long-form roadmap history that used to live inside `roadmap.md`.

Use `roadmap.md` for current direction. Use these files only when an agent needs
historical evidence, implementation notes, or old acceptance details.

## Files

- `2026-06-18-roadmap-pre-split.md` — full pre-split roadmap snapshot. This is
  the lossless archive of all historical sections moved out of the active
  roadmap.

## Reading Rule

Historical entries are not current instructions unless the active `roadmap.md`,
`RUNBOOK.md`, or a current decision document explicitly points to them.
